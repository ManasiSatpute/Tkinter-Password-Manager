#  MyPass – Tkinter Password Manager

MyPass is a simple desktop password manager application built using **Python and Tkinter**.  
It helps users generate strong random passwords and store login credentials locally through an easy-to-use graphical interface.

---

##  Why This Project?

Managing multiple passwords is difficult and insecure when done manually.  
This project demonstrates how a GUI-based Python application can automate password generation and safely organize credentials at a basic level.

---

##  Features

- Clean and minimal Tkinter-based GUI  
- Random password generator with letters, numbers, and symbols  
- Save website credentials locally  
- Input validation with confirmation dialogs  
- Modular code structure for better maintainability  

---

##  Tech Stack

- **Language:** Python 3  
- **GUI Library:** Tkinter  
- **Core Concepts:** File handling, modular programming, user input validation
